Computo Metrico GIS

Plugin QGIS per trasformare geometrie, codici voce e prezziari in un computo metrico estimativo completo, con archivio SQLite, catalogo regionale, contabilità lavori, report PDF/XLSX e progetto demo.

QGIS 3.x QGIS 4.99 ready Qt6 aware SQLite PDF + XLSX + Mappa

Workflow

  • Inizializza il database locale del plugin.
  • Usa il catalogo regionale per capire subito cosa è pronto, scaricabile o manuale.
  • Importa un dataset incluso, un file CSV/XLSX o un DCF leggibile.
  • Calcola quantità da punti, linee e poligoni con campi di precisione opzionali.
  • Gestisce SAL, certificato di pagamento e giornale lavori nella scheda Contabilità.
  • Genera PDF e XLSX con logo, intestazione, riferimenti e mappa corrente.

Contesto locale

Versione plugin: __VERSION__

Directory dati: __DATA_DIR__

Database SQLite: __DB_PATH__

Prezziari caricati: __PRICE_LISTS__

Run di computo salvati: __RUNS__

Catalogo e DCF

Il plugin distingue tra dataset già normalizzati, file ufficiali solo consultabili e regioni ancora da convertire. Se il formato ufficiale non è pronto, crea un template manuale già compatibile col parser.

I file .dcf sono supportati solo quando il contenitore espone dati leggibili come ZIP, SQLite o tabella uniforme. Se il DCF è proprietario, il plugin lo segnala senza importare dati ambigui.

Formato prezziario

__FORMAT_HELP__

Se il plugin rileva colonne o valori non uniformi, mostra all’utente il formato corretto e lascia comunque accessibili i file scaricati nella cartella locale.
